## v1.0.1

### What's New

**🔧 Last.fm Scrobbling**
You can now connect your Last.fm account inside the app. Go to **Settings → Last.fm Integration**, log in with your username and password, and your listens will be scrobbled automatically.

**🔄 In-App Updates (Now for Everyone!)**
The **System Update** button now works on all builds — no more being sent to a browser. Tap it to check for new versions, see what's new, and download/install directly inside the app.

**🖼️ Cleaner About Screen**
- Removed the red tint from the app logo — shows the actual logo art now.
- Added quick links to the GitHub repo and Buy Me a Coffee.

**🧹 Appearance Settings**
Removed the old **Legacy Icon** toggle — one less option you don't need.

### Bug Fixes
- Fixed playback errors that could crash the app when changing tracks.
- Fixed slow lyrics loading on some songs.
- Fixed various smaller issues.

### Download

Choose the right APK for your device:

| File | Best for |
|------|----------|
| `app-universal-foss-release.apk` | **Most users** — works on any device (largest file) |
| `app-arm64-foss-release.apk` | **Modern phones (2020+)** — smaller download |
| `app-armeabi-foss-release.apk` | **Older phones** — rare, only if universal doesn't work |
| `app-x86-foss-release.apk` | **Tablets / Emulators** |
| `app-x86_64-foss-release.apk` | **Newer tablets / Emulators** |

**Not sure which to pick?** Download `app-arm64-foss-release.apk` if your phone is from 2020 or newer. Otherwise grab `app-universal-foss-release.apk`.
